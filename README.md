# LUXHT Fix Website

Professional property maintenance and improvement services website for LUXHT Fix, serving Orlando, Central Florida, and South Florida.

## 🚀 Tech Stack

- **Framework**: Next.js 16 (App Router)
- **Styling**: Tailwind CSS 4
- **UI**: Lucide React icons, Framer Motion animations
- **Deployment**: Static export for Cloudflare Pages

## 📦 Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## 🌐 Deployment to Cloudflare Pages

The site is configured for static export. After building, the `out/` directory contains the complete static site.

### Quick Deploy Steps:
1. Push to GitHub
2. Connect to Cloudflare Pages
3. Set build command: `npm run build`
4. Set output directory: `out`

See detailed instructions in the deployment guide.

## 📁 Project Structure


## 🎨 Features

- **Transparent Pricing**: Published cost guides for all services
